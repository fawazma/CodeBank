### Instructions:

- Compile:
```
g++ -std=c++11 *.cpp -o bst_database.out
```
- Run:
```
./bst_database.out 
```
- During run time, all operations like addition/deletion
are in memory, but while exiting, the remaining data will
be saved to files.
